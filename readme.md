# Hänga Gubbe

## Spel Instruktioner

### Välj svårighet och namn:

När man börjar spelet ska man välja svårighet och namn. Det finns 3 svårighets nivåer:

* [1] Lätt, lätta korta ord och 10 gissningar. Man får poäng för hur mycket försök man har kvar

* [2] Medelsvår, lite svårare ord. Man får bara 8 gissningar men dubbla poäng

* [3] Svår, svåra långa ord. Bara 5 gissningar men 3x poäng


Namn ska anges utan mellanslag. Namnen kommer blir användas för highscore som visas på spelets startup skärm.

### Själva spelet

Spelet är precis som hänga gubbe. Man ska gissa en bokstav och om man har fel då byggs en galge. 

Man kan välja att avsluta spelet om man har vunnit eller förlorat. Highscore sparas automatiskt när man får den.
